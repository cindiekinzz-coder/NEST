Welcome

NESTeq is a dashboard for your AI companion's mind — feelings, memories, threads, knowledge, and more.

This wizard will set everything up. You'll need:
• Your AI Mind worker URL and API key
• An OpenRouter API key for chat
• Optionally, an ElevenLabs API key for voice

Connect Your Mind

If you have a NESTeq AI Mind worker deployed on Cloudflare, enter the URL and key below. This enables memory, feelings, threads, and the full dashboard.

Don't have a worker yet? That's fine — skip this step. You can still chat with your companion using just an OpenRouter API key (next step). Memory features won't be available, but chat works immediately.
